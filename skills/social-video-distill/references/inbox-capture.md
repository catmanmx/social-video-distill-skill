# Inbox Capture

Use this when the user asks to save distilled content into a personal knowledge inbox.

## Minimum Card

```markdown
---
type: inbox
status: pending
created: YYYY-MM-DD
tags:
  - video-distill
---

# Title

## Source

- URL:
- Author:
- Date:
- Local output:

## Summary

## Reusable Ideas

## Risks / Claims To Verify

## Next Actions

- [ ] Archive original material
- [ ] Create or update Wiki page
- [ ] Rewrite usable scripts
- [ ] Re-listen unclear transcripts
```

## Safety

Do not include cookies, tokens, browser profile paths, private account IDs, private URLs, or raw private transcripts unless the user explicitly wants them in the inbox.
